# 고씨네 떡볶이 - 데이터 기반 무역 파트너

> 첫 수출 기업을 위한 데이터 기반 무역 파트너
> 복잡한 시황 분석부터 서류 작성까지, 무역의 시작을 체계적으로 지원합니다.

---

## 📁 프로젝트 구성

| 폴더 | 설명 | 문서 |
|------|------|------|
| [1.mail/](./1.mail/) | AI 무역 인콰이어리 메일 분석 및 답장 생성 시스템 | [README](./1.mail/README.md) |
| [2.ship/](./2.ship/) | 글로벌 공급망 디지털 트윈 - 무역 인텔리전스 대시보드 | [README](./2.ship/README.md) |
| [3.invoice/](./3.invoice/) | AI 기반 국제무역 자동화 플랫폼 (HS CODE ~ Proforma Invoice) | [README](./3.invoice/README.md) |
| [4.homepage/](./4.homepage/) | 서비스 랜딩 페이지 | - |

## 🛠️ 기술 스택

| 분류 | 기술 |
|------|------|
| **Backend** | Python 3.10+, Streamlit |
| **AI** | OpenAI GPT |
| **Database** | SQLite3 |
| **API** | Gmail API, KITA Open API |
| **Frontend** | HTML/CSS/JS (Tailwind CSS) |

---

## 🚀 실행 방법

각 모듈별 실행 방법은 해당 폴더의 README를 참고하세요.

```bash
# 메일 분석 시스템
cd 1.mail && streamlit run main_final.py

# 선박 대시보드
cd 2.ship && streamlit run app.py

# 인보이스 시스템
cd 3.invoice && streamlit run app.py
```
